# video-player

![Deploy](https://github.com/dalyIsaac/video-player/workflows/Deploy/badge.svg)

This is a PWA for playing local video files.

## Building

Prerequisites:

- git
- Node 12.x
- Yarn

```shell
git clone https://github.com/dalyIsaac/video-player.git
cd video-player
yarn
```

To run:

```shell
yarn start
```
